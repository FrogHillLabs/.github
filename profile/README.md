# Frog Hill Labs

**Practical technology, built close to home.**

Frog Hill Labs is a small independent workshop in northern Wisconsin for software, radio, automation, local-first systems, electronics, and useful experiments at the edge of the woods and water.

## What happens here

- **Radio and field sensing** — LoRa, Meshtastic, SDR, remote sensors, and practical ways to move a small amount of useful information a long way.
- **Local-first systems** — Self-hosted services, resilient home infrastructure, automation, and private systems designed to remain useful without a cloud dependency.
- **Software and hardware** — Practical software, electronics repair, prototypes, reverse engineering, and small tools that make existing systems clearer, sturdier, or more capable.

## Working principles

- Useful over flashy
- Understandable over opaque
- Local-first where practical
- Field-tested before polished
- Document the path, not only the result

## Current projects
